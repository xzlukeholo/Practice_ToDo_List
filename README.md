# Python To-Do List Practice

A small command-line to-do list app I made while practicing Python basics.

The app lets users add, view, and delete to-do items. I also added input validation so the program can handle wrong input and empty tasks.

This project helped me review:

- while loops
- if / elif / else
- lists
- append() and pop()
- try / except
- strip()
- truthy and falsy values
